1.Apa yang dimaksud dengan J2EE? (POIN: 10)
Jawab : J2EE merupakan sebuah edisi platform dari bahasa pemrograman Java yang berstandar untuk menyelesaikan komplesitas masalah pada perusahaan. J2EE menyediakan API (Application Program Interface) dan Run Time Environment untuk mengembangkan dan menjalankan perangkat lunak multiple network yang dapat menggunakan web service, secure network application, dan aplikasi skala besar lainnya. J2EE singkatan dari Java 2 Enterprise Edition dan semenjak versi 5 berubah menjadi java EE atau Java Enterprise Edition. Java Enterprise Edition adalah sebuah spesifikasi sehingga ada banyak vendor software/ komunitas opensource membuat software application server yang mengikuti standar Java Enterprise Edition. J2EE sendiri merupakan sebuah edisi platform dari bahasa pemrograman Java yang berstandar untuk menyelesaikan komplesitas masalah pada perusahaan. J2EE menyediakan API (Application Program Interface) dan Run Time Environment untuk mengembangkan dan menjalankan perangkat lunak multiple network yang dapat menggunakan web service, secure network application, dan aplikasi skala besar lainnya. Beberapa API yang disediakan oleh J2EE diantaranya adalah API untuk berkomunikasi dengan web service dan untuk mengembangkan aplikasi berbasis web.
